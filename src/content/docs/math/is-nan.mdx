```lua
math.isnan(x)
```

Checks if a value is NaN (Not a Number).

### Parameters

| Parameter | Type     | Required | Description         |
| --------- | -------- | -------- | ------------------- |
| x         | `number` | Yes      | The value to check. |

### Return Values

| Type      | Description                                    |
| --------- | ---------------------------------------------- |
| `boolean` | `true` if the value is NaN, `false` otherwise. |

### Examples

```lua
math.isnan(69514) -- false
math.isnan(math.nan) -- true
```
