description: math.isinf()
---

```lua
math.isinf(x)
```

Checks if the given number `x` is infinite.

### Parameters

| Parameter | Type     | Required | Description          |
| --------- | -------- | -------- | -------------------- |
| x         | `number` | Yes      | The number to check. |

### Return Values

| Type      | Description                                          |
| --------- | ---------------------------------------------------- |
| `boolean` | `true` if the number is infinite, `false` otherwise. |

### Examples

```lua
math.isinf(math.huge) -- true
math.isinf(101969285) -- false
```
