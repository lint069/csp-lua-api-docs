---

```lua
math.isfinite(x)
```

Checks if the given number `x` is finite (not infinite or NaN).

### Parameters

| Parameter | Type     | Required | Description          |
| --------- | -------- | -------- | -------------------- |
| x         | `number` | Yes      | The number to check. |

### Return Values

| Type      | Description                                        |
| --------- | -------------------------------------------------- |
| `boolean` | `true` if the number is finite, `false` otherwise. |

### Examples

```lua
math.isfinite(101969285) -- true
math.isfinite(math.huge) -- false
math.isfinite(math.nan) -- false
```
