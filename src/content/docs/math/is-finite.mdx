## Signature

```lua
math.isfinite(x)
```

## Description

Checks if a number is finite (not infinity and not NaN).

## Parameters

| Parameter | Type     | Required | Description          |
| --------- | -------- | -------- | -------------------- |
| x         | `number` | Yes      | The number to check. |

## Return Values

| Type      | Description                                        |
| --------- | -------------------------------------------------- |
| `boolean` | `true` if the number is finite, `false` otherwise. |

## Examples

```lua
math.isfinite(101969285) -- true
math.isfinite(math.huge) -- false
```
