### Signature

```lua
math.tau
```

### Description

A constant equal to 2π (~6.28). Equivalent to writing `math.pi * 2`. Useful for working with full rotations in radians.

### Examples

```lua
ui.pathArcTo(vec2(50, 50), 20, 0, math.tau)
ui.pathStroke(rgbm.colors.white)
```
