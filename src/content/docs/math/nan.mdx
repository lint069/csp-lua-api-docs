## Signature

```lua
math.nan
```

## Description

A constant representing NaN (Not a Number). NaN is the result of undefined numeric operations like `0 / 0`.
NaN is never equal to itself or any other number. Use [`math.isnan()`](/math/is-nan/) to check for it.

## Examples

```lua
local nan = 0 / 0

ac.debug('nantest', nan ~= nan) -- true
```
