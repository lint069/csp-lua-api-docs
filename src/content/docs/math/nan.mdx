description: math.nan
---

```lua
math.nan
```

NaN (Not a Number) is the result of undefined numeric operations like `0 / 0`.
NaN is never equal to itself or any other number. Use [`math.isnan()`](/math/is-nan/) to check for it.

### Examples

```lua
local nan = 0 / 0

ac.debug('nantest', nan ~= nan) -- true
```
