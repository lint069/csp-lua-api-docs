```lua
io.exists(filename)
```

Checks if file or directory exists.
If you need to know specifically if a directory or file exists, use [`io.dirExists()`](/io/dir-exists/) or [`io.fileExists()`](/io/file-exists/) instead.

### Parameters

| Parameter | Type     | Required | Default | Description                                 |
| --------- | -------- | -------- | ------- | ------------------------------------------- |
| filename  | `string` | Yes      | -       | The path to the file or directory to check. |

### Return Values

| Type      | Description                                                |
| --------- | ---------------------------------------------------------- |
| `boolean` | `true` if the file or directory exists, `false` otherwise. |

### Examples

```lua
if io.exists("./file.txt") then
    ac.log("Exists!")
end
```
