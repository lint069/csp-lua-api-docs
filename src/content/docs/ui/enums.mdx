This reference lists the enumerations used as parameters across various `ui` functions.

## `ui.CornerFlags`

Used to determine which corners of a shape should be rounded. These are bitmasks and can be combined using `bit.bor()`.

| Flag                         | Value | Description                  |
| :--------------------------- | :---- | :--------------------------- |
| `ui.CornerFlags.None`        | `0`   | No rounding.                 |
| `ui.CornerFlags.TopLeft`     | `1`   | Round top-left corner.       |
| `ui.CornerFlags.TopRight`    | `2`   | Round top-right corner.      |
| `ui.CornerFlags.BottomLeft`  | `4`   | Round bottom-left corner.    |
| `ui.CornerFlags.BottomRight` | `8`   | Round bottom-right corner.   |
| `ui.CornerFlags.Top`         | `3`   | Round both top corners.      |
| `ui.CornerFlags.Bottom`      | `12`  | Round both bottom corners.   |
| `ui.CornerFlags.Left`        | `5`   | Round both left corners.     |
| `ui.CornerFlags.Right`       | `10`  | Round both right corners.    |
| `ui.CornerFlags.All`         | `15`  | Round all corners (default). |
