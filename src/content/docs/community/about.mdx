This documentation site is a community-driven initiative to provide an exhaustive,
organized, and searchable reference for the Custom Shaders Patch (CSP) Lua SDK.

## Disclaimer

This project is not an official Kunos Simulazioni or Custom Shaders Patch publication.
It is maintained by developers for developers.
Information provided is based on the SDK meta-definitions and community testing.
